## üìã Writing Guidelines

### Chapter Structure
Each chapter should follow the established format:

```markdown
# Chapter [Number]: [Title]

[Opening scene/paragraph]

---

[Chapter content with appropriate scene breaks marked by ---]

---

[Closing paragraph/cliffhanger]
```

### POV Guidelines
- **Main chapters (39-50)**: Riven POV by default
- **Interstitial chapters (.5)**: Raechelle POV
- Maintain consistent voice for each character
- Use established narrative style from Books 1-3

### Length Guidelines
- Target: 5,000-6,500 words per chapter
- Allow flexibility for natural chapter breaks
- Interstitial chapters can be slightly shorter (4,000-5,000 words)

### Continuity Checklist
Before writing each chapter, verify:
- [ ] Character states match previous chapters
- [ ] Timeline is consistent with Book 4 outline
- [ ] Magic system usage aligns with established rules
- [ ] Character relationships reflect current arc status
- [ ] References to Books 1-3 events are accurate

## üîÑ Workflow

### Writing Process
1. **Planning**: Review Book4.md outline for chapter goals
2. **Drafting**: Write chapter in this directory
3. **Review**: Self-edit for consistency and quality
4. **Integration**: Once polished, chapter will be integrated into main narrative
5. **Update**: Update Book4.md with completion status

### File Naming Convention
- Main chapters: `Chapter-[#].md`
- Interstitial chapters: `Chapter-[#.5]-[POV-Character]-POV.md`
- Use zero-padding if needed (e.g., `Chapter-39.md`)

### Version Control
- Commit chapters individually as they're completed
- Include meaningful commit messages
- Reference chapter number and title in commit
