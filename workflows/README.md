# Workflows Directory

This directory contains documentation about development workflows and processes for the BloodCraft project.

## Available Documents

### `WORKFLOW.md`
Comprehensive guide to the development workflow including:
- How to contribute to the project
- Git workflow and branching strategy
- Review and merge processes
- Parallel development guidelines

### `WORKFLOW_DIAGRAM.txt`
Visual ASCII diagram showing the development workflow:
- Step-by-step process flow
- Decision points
- Integration paths

### `QUICK_FIX_GUIDE.md`
Quick reference for common fixes and tasks:
- Fast solutions to common issues
- Troubleshooting tips
- Quick commands and shortcuts
